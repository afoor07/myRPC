# myRPC - Remote Procedure Call System for Astra Linux

## Описание

myRPC - это система удаленного вызова команд, реализованная на языке C для операционной системы AstraLinux SE 1.7. Система состоит из двух компонентов:

- **myRPC-client** - консольная утилита для отправки команд на удаленный сервер
- **myRPC-server** - сервер-демон, выполняющий команды от авторизованных пользователей

## Архитектура

└─────────────┘         └─────────────┘
│                       │
│ ┌─────────────────────┤
│ │ Text/JSON Protocol  │
│ │ User Authentication │
│ │ Command Execution   │
│ └─────────────────────┘

## Требования к сборке

- ОС: AstraLinux SE 1.7
- Компилятор: GCC 4.8+
- Зависимости: libjson-c-dev, systemd
